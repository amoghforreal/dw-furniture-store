# DW Furniture Store

A modern, responsive e-commerce application designed for a seamless furniture shopping experience.

## Project Overview

This application provides a complete user journey from authentication to product selection and simulated checkout. It features a clean, minimalist design focused on usability and visual appeal.

## Key Features

* **Authentication**: Secure Login and Signup functionality with Guest access support.
* **Product Catalog**: Dynamic product listing with sorting (Price, A-Z) and filtering capabilities.
* **Product Details**: comprehensive product views with image galleries, quantity adjusters, and stock status.
* **Shopping Cart**: Real-time cart management, total calculation, and item removal.
* **User Profile**: Dashboard to manage user details, viewing security settings, and addresses.
* **Responsive Design**: Optimized for various screen sizes.

## Tech Stack

* **Frontend**: React.js
* **Styling**: Tailwind CSS
* **Routing**: React Router DOM
* **Icons**: Lucide React
* **Build Tool**: Vite

## Getting Started

1.  **Clone the repository**
    ```bash
    git clone [https://github.com/amoghforreal/dw-furniture-store.git]
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Run the development server**
    ```bash
    npm run dev
    ```
